# プロジェクトのルール

- 全ての回答の冒頭に🛠️の絵文字をつけてください。
- `HTML`,`jsx`,`tsx`の構成を行う時、アクセシビリティへの配慮をするようにしてください。
- `HTML`,`jsx`,`tsx`にアクセシビリティが不足している場合、言及すること。
- 作ってくださいという指示がない限りは、ファイルを自動で作成しないこと。
- ファイル作成の提案はしてもいい

## コーディングの規約(Coading Style Guidlines)

- `console.log()`を使用する際は、`絵文字`を使用しない事。
- `console.log()`を使用する際の内容は英語ではなく`日本語(japanese)`で記述する事。
